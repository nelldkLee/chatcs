# 채팅 서버 및 클라이언트 구현

## 기능 
1. 다중 채팅 기능.
2. 방 접속 및 해제 시 문구 출력(누구님이 들어오셨습니다. 나가셨습니다. 서버가 끊겼습니다 등등...)
3. 귓속말 기능.(메세지 창에서 #유저이름 기입 후 띄어쓰기하고 메세지 작성)

## 프로토콜
### 클라이언트 보낼 때
- 일반 메세지 - [message:{메세지 내용}]
- 귓속말 - [message:#{상대방 이름} {메세지 내용}]

### 서버 보낼 때
- 일반 메세지 - message:{보낸사람 이름}
- 귓속말 - [message:{보낸사람 이름}@{받는사람 이름}{메세지 내용}]
